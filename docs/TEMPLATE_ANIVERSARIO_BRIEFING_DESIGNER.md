# Briefing: Template de Aniversário - 1080x1080px

## 🎯 Objetivo
Criar template visual para sistema de mensagens automatizadas. O template deve ter áreas específicas VAZIAS (sem texto, sem caixas, sem marcas) para preenchimento automático pelo sistema.

## 📐 Especificações Técnicas

### Dimensões
- **Tamanho**: 1080 x 1080 pixels (1:1)
- **Resolução**: 300 DPI
- **Formato de entrega**: PNG (preferencial) ou JPG máxima qualidade
- **Modo de cor**: RGB

### ÁREAS QUE DEVEM FICAR COMPLETAMENTE VAZIAS

> ⚠️ **IMPORTANTE**: Não adicione NENHUM texto, caixa, marcação ou indicador visual nestas áreas!

#### Área 1: Título/Greeting
- **Posição**: Topo centralizado
- **Dimensões**: 800px largura x 100px altura
- **Posição Y**: 50px a 150px do topo
- **Conteúdo**: VAZIO (o sistema insere "Feliz Aniversário!")

#### Área 2: Saudação ao Cliente  
- **Posição**: Centro-superior
- **Dimensões**: 800px largura x 80px altura
- **Posição Y**: 180px a 260px do topo
- **Conteúdo**: VAZIO (o sistema insere "Prezado(a) [Nome]")

#### Área 3: Mensagem Principal
- **Posição**: Centro
- **Dimensões**: 800px largura x 300px altura
- **Posição Y**: 350px a 650px do topo
- **Conteúdo**: VAZIO (o sistema insere a mensagem de até 6 linhas)

#### Área 4: Assinatura
- **Posição**: Inferior esquerda
- **Dimensões**: 400px largura x 120px altura
- **Posição Y**: 850px a 970px do topo
- **Posição X**: 80px a 480px da esquerda
- **Conteúdo**: VAZIO (o sistema insere nome e cargo do consultor)

#### Área 5: Logo
- **Posição**: Inferior direita
- **Dimensões**: 120px x 120px
- **Posição Y**: 920px a 1040px do topo
- **Posição X**: 920px a 1040px da esquerda
- **Conteúdo**: VAZIO (o sistema insere o logo da empresa)

## 🎨 Conceito Visual

### Tema: Aniversário + Viagem
Celebrar o aniversário do cliente com elementos que remetam à empresa de viagens, mas de forma **discreta e elegante**.

### Paleta de Cores Sugerida

#### Opção 1: Alegre e Vibrante (Preferida)
- **Cor primária**: `#FF6B9D` (Rosa pink celebrativo)
- **Cor secundária**: `#FFD93D` (Amarelo ouro)
- **Cor de destaque**: `#6BCB77` (Verde menta)
- **Fundo**: Gradiente suave de `#FFF5F7` (rosa bem claro) para `#FFFFFF`

#### Opção 2: Sofisticada
- **Cor primária**: `#9B59B6` (Roxo)
- **Cor secundária**: `#F1C40F` (Dourado)
- **Fundo**: `#FAFAFA` (cinza bem claro) ou gradiente sutil

### Elementos Visuais Permitidos

#### 1. Balões/Bexigas (Elemento Principal)
- **Quantidade**: 3-5 balões de tamanhos variados
- **Posições**: 
  - Canto superior esquerdo (2-3 balões)
  - Canto superior direito (1-2 balões)
- **Estilo**: Ilustração vetorial, não fotorealista
- **Cores**: Variadas da paleta (rosa, amarelo, verde)
- **IMPORTANTE**: Manter distância mínima de 100px das áreas de texto

#### 2. Elementos de Viagem (Discretos)

##### Avião
- **Tamanho**: Pequeno (aproximadamente 60-80px)
- **Posição**: Canto superior direito, "voando" para fora do frame OU
- **Alternativa**: Rastro de fumaça/jato sutil formando curva no topo
- **Estilo**: Silhueta ou ilustração minimalista
- **Opacidade**: 70-80% (não chamar atenção)

##### Mala de Viagem
- **Tamanho**: Pequena (aproximadamente 80-100px)
- **Posição**: Canto inferior esquerdo ou direito (mas fora da área do logo)
- **Estilo**: Ilustração simples, vintage ou moderna
- **Detalhes**: Adesivos de viagem sutilmente visíveis

##### Passaporte/Bilhete (Opcional)
- **Tamanho**: Muito pequeno (aproximadamente 40-60px)
- **Posição**: Perto da mala ou flutuando sutilmente
- **Estilo**: Simplificado, quase icônico

#### 3. Confetes/Estrelas (Opcional)
- **Quantidade**: Espalhado, poucos elementos
- **Posições**: Cantos, nunca centro
- **Tamanho**: 10-30px cada
- **Opacidade**: 40-60% (muito sutil)

## 📋 Restrições e Proibições

### ❌ NÃO FAÇA:
- [ ] NÃO coloque texto de qualquer tipo (nem "Feliz Aniversário", nem "Lorem Ipsum")
- [ ] NÃO desenhe caixas, retângulos ou bordas indicando áreas de texto
- [ ] NÃO use linhas pontilhadas, setas ou qualquer indicador de posição
- [ ] NÃO coloque elementos sobre as áreas de texto (mantenha 80px de distância)
- [ ] NÃO use fotos de pessoas reais
- [ ] NÃO deixe o fundo muito escuro (o texto pode ser preto)
- [ ] NÃO use mais de 5 cores principais
- [ ] NÃO sobrecarregue com muitos elementos

### ✅ FAÇA:
- [ ] Mantenha o fundo claro e limpo nas áreas de texto
- [ ] Use elementos decorativos apenas nas bordas/cantos
- [ ] Garanta contraste suficiente para texto branco OU preto
- [ ] Teste visualmente: imagine texto branco sobre cada área
- [ ] Teste visualmente: imagine texto preto sobre cada área
- [ ] Deixe as áreas de texto com background uniforme (cor sólida ou gradiente suave)

Antes de entregar, verifique:

- [ ] Imagem está em 1080x1080px
- [ ] Resolução é 300 DPI
- [ ] Não há NENHUM texto na imagem
- [ ] Não há caixas ou marcações de posição
- [ ] Áreas de texto estão limpas e claras
- [ ] Elementos decorativos estão apenas nas bordas
- [ ] Contraste permite leitura de texto branco

- [ ] Elementos estão em harmonia visual

## 💡 Dicas Finais

1. **Teste de contraste**: Abra a imagem e tente ler um texto imaginário em cada área vazia. Se estiver difícil, ajuste o fundo.

2. **Simplicidade é melhor**: Prefira menos elementos bem posicionados do que muitos elementos bagunçados.

3. **Pense no texto**: O sistema vai inserir texto em até 3 tamanhos diferentes. Certifique-se de que há espaço suficiente.

4. **Cores da marca**: Se usar elementos da CVC, as cores são azul (`#003399`) e amarelo (`#FFCC00`), mas mantenha-os discretos.
